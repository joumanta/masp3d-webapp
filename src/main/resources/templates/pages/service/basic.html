홈 서비스 Basic

개요

Basic

금형 및 제품 설계시 게이트 선정, 성형성 검토를 위한 기초 해석을 기준

상세기능

  • Studio

    MAPS-3D Studio는 사출성형 해석에 필요한 재질 및 성형 조건을 설정하여 작업 할 수 있도록 도와주며, 다양하고 편리한 방법으로 쉽게 결과 분석 작업을 수행 할 수 있는 GUI환경을 제공합니다. 해석 결과를 Contour, Shading, XY Plot 타입으로 보여주며, Animation 및 Section Display등 다양한 해석 결과 구현이 가능합니다.

  • Modeler

    MAPS-3D Modeler는 3D CAD/CAE 시스템과의 완벽한 Interface를 제공하며, 초보자도 어려움 없이 사용할 수 있도록 쉬운 GUI환경을 제공합니다.

  • Flow

    사용한 시간만큼만 요금이 과금되며 서비스 운영에 따라 서버를 투입/반납 및 사양 변경할 수 있어 탄력적으로 비용을 관리할 수 있습니다.

  • Insert

    양질의 성형품을 얻기 위하여 제품 및 금형 설계/사출 성형 공정을 최적화하는 가장 기본적인 해석 모듈이며, 사출 성형 과정 중 용융된 수지가 금형 내로 충전되는 과정을 해석하는 모듈입니다.

  • Material D/B
    • - Thermoplastics Materials
    • - Thermoset Materials
    • - Mold Materials
    • - Coolant Materials
    • - Injection Machines
    • - Insert Materials
    • - Gas Materials

Studio


  • - MAPS-3D Viewer : 라이센스 없이도 해석 결과를 확인 할 수 있도록 Viewer 무상 제공
  • - Project Wizard : 해석용 프로젝트 생성 및 성형 조건 입력 등을 초보자도 사용할 수 있도록 도와주는 편의기능
  • - Analysis Manager : Batch Job을 통한 해석 작업 효율을 높여주는 기능
  • - Edit Project contents : 신규 및 기존 Project들의 조건을 쉽게 변경 할 수 있는 편의 기능
  • - Detail Chart : 해석 결과의 세부 정보를 한눈에 확인 할 수 있는 편의 기능
  • - Bookmark : 사용자가 원하는 화면 정보를 저장하여 언제든 사용 할 수 있는 기능
  • - Report Wizard : HTML / PPT 형태로 보고서 자동 생성하는 기능

Modeler


  • - CAD interface : IGES, STEP, CATIA V4, CATIA V5, Pro-E Inventor, VDA, ACIS, STL
  • - FEA interface : Nastran, Universal file, LS-DYNA
  • - Geometry의 수정 및 편집 기능
  • - Runner, Cooling channel system 생성 기능
  • - 요소(Mesh)의 수정과 편집을 쉽고 빠르게 구현할 수 있는 다양한 기능
  • - Surface meshing기능 및 요소(Mesh) 품질을 자동으로 향상 해주는 Remeshing기능
  • - Key nodes, Local mesh 등 복잡한 제품 형상에서 발생되는 요소 품질 저하 현상을 쉽게 예방할 수 있는 편의 기능
  • - Gradient Offset(자동으로 빼기 구배 부여 기능) 등과 같이 요소(Mesh) 상태에서 형상 변경이 가능한 기능

Flow


  • - 시간에 따른 유동 선단의 진행 모습 가시화
  • - 제품 성형에 필요한 사출압 및 형체력 계산
  • - 제품의 살 두께에 따른 영향과 게이트, 런너 시스템의 치수 및 위치 최적화
  • - 미성형(Short shot), 웰드라인(Weld line), 에어트랩(Air trap) 예측
  • - 충전시간, 사출온도, 금형 온도 등의 사출 조건 최적화
  • - Balanced Flow에 의한 과보압(Over-packing) 및 아보압(Under-packing) 예방
  • - 다단 사출을 통한 제품 외관의 균일화
  • - 핫 런너(Hot runner) 시스템의 유효성 분석 및 방안 제시
  • - Hot Runner Temperature Control 기능을 이용하여 핫 런너 각각의 온도 설정 가능
  • - Mold Temperature Profile 기능을 이용하여 제품의 영역별 금형 온도 및 시간에 따른 온도 설정 가능(급속가열 / 급속냉각 등에 응용)
  • - MAPS-3D Flow는 Fast Filling, Filling, Compressible Filling 등으로 세분화 되어 사용자의 목적에 맞는 솔루션을 제공함으로써 현업에서 발생되는 변화에 신속하게 대응할 수 있음
  • - 해석용 컴퓨터의 발전에 따라 병렬연산(Parallel computing) 기법을 기본 제공(Hexa-core)

Insert


  • - 금속, 유리, 열가소성 수지 및 열경화성 수지와 같이 사출 성형 중에 사용되는 모든 인서트를 고려할 수 있음
  • - 변형 해석시 Constraint node를 설정함으로써, 사용자가 원하는 변형의 기준을 설정할 수 있음
  • - 인서트의 초기온도(Pre-heating)를 부여함으로써, 인서트의 온도 조건에 따른 제품의 물리적 변화 예측
